# Contexto Técnico Inicial - DevToolsPro

## Módulos existentes
- `quality-gate-service`: consolida resultados de análise estática.
- `build-orchestrator`: coordena execução de pipelines internos.
- `dev-portal`: interface para times acompanharem builds e qualidade.

## Stack predominante
- Backend: Node.js
- Frontend: React
- Persistência: PostgreSQL
- Mensageria: RabbitMQ

## Desafios atuais
- README técnico desatualizado para novos módulos.
- Falta de padrão na descrição de variáveis de ambiente.
- Time de suporte não possui fluxo único de troubleshooting.
